# Syed Sameed — Professional Portfolio
A four-page personal portfolio site built to support your resume and showcase your full engineering background.

## Pages
- `index.html` — About Me (hero, stats, bio, education, skills)
- `projects.html` — All Projects (PCB, Robotic Arm, Soft Robotics, MEMS, THV Report)
- `capstone.html` — LEA 2.0 Capstone Deep-Dive
- `contact.html` — Contact page with links and availability

## How to Deploy on GitHub Pages

1. **Create a repo named** `yourusername.github.io` on github.com (Public, no template)
2. **Upload all 5 files**: `index.html`, `projects.html`, `capstone.html`, `contact.html`, `style.css`
3. Go to **Settings → Pages → Deploy from branch → main → / (root) → Save**
4. Visit `https://yourusername.github.io` in ~1 minute

## To personalise
- Add your GitHub and LinkedIn URLs in `index.html` and `contact.html`
- Add a headshot photo when you have one — replace the badge-stack area in `index.html`
- Add photos of your actual projects (PCB renders, robot photos) to replace the SVG icons in `projects.html`

## Design
- **Fonts**: Playfair Display (headings) · DM Mono (labels) · DM Sans (body)
- **Accent**: `#c8a96e` warm gold
- **Theme**: Dark editorial
- **No frameworks** — pure HTML/CSS/JS, no build step needed
